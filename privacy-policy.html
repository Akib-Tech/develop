Please keep in mind that if you directly disclose personally identifiable information or personally sensitive data through Boca Bay DUI Program public message boards,this information may be collected and used by others. Note: Boca Bay DUI Program does not read any of your private online communications.

Boca Bay DUI Program encourages you to review the privacy statements of Web sites you choose to link to from Boca Bay DUI Program so that you can understand how those Web sites collect, use and share your information. Boca Bay DUI Program is not responsible for the privacy statements or other content on Web sites outside of the Boca Bay DUI Program and Boca Bay DUI Program family of Web sites.

Use of Cookies

The Boca Bay DUI Program Web site use “cookies” to help you personalize your online experience. A cookie is a text file that is placed on your hard disk by a Web page server. Cookies cannot be used to run programs or deliver viruses to your computer.Cookies are uniquely assigned to you, and can only be read by a web server in the domain that issued the cookie to you.

One of the primary purposes of cookies is to provide a convenience feature to save you time. The purpose of a cookie is to tell the Web server that you have returned to a specific page. For example, if you personalize Boca Bay DUI Program pages, or register with Boca Bay DUI Program site or services, a cookie helps Boca Bay DUI Program to recall your specific information on subsequent visits. This simplifies the process of recording your personal information, such as billing addresses, shipping addresses, and so on. When you return to the same Boca Bay DUI Program Web site, the information you previously provided can be retrieved, so you can easily use the Boca Bay DUI Program features that you customized.

You have the ability to accept or decline cookies. Most Web browsers automatically accept cookies, but you can usually modify your browser setting to decline cookies if you prefer. If you choose to decline cookies, you may not be able to fully experience the interactive features of the Boca Bay DUI Program services or Web sites you visit.

Security of your Personal Information

Boca Bay DUI Program secures your personal information from unauthorized access, use or disclosure. Boca Bay DUI Program secures the personally identifiable information you provide on computer servers in a controlled, secure environment, protected from unauthorized access, use or disclosure. When personal information (such as a credit card number) is transmitted to other Web sites, it is protected through the use of encryption, such as the Secure Socket Layer (SSL) protocol.
